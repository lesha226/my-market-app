# my-market-app
Учебный проект 5 спринта для курса мидл java-разработчик, яндекс практикум


## Web API

| URL                                   | Method  | Description                                                                    | Optional params                            |
|---------------------------------------|---------|--------------------------------------------------------------------------------|--------------------------------------------|
| `/items`, `/`                         | `GET`   | Эндпоинт получения товаров на странице                                         | `search`, `sort`, `pageNumber`, `pageSize` | 
| `/items?id=[id]&action=[action]`      | `POST`  | Эндпоинт уменьшения/увеличения количества товара в корзине со страницы товаров | `search`, `sort`, `pageNumber`, `pageSize` |  
| `/items/{id}`                         | `GET`   | Эндпоинт получения страницы с товаром                                          |                                            |
| `/items/{id}?action=[action]`         | `POST`  | Эндпоинт уменьшения/увеличения количества товара в корзине со страницы товара  |                                            |
| `/cart/items`                         | `GET`   | Эндпоинт получения страницы со списком товаров в корзине                       |                                            |
| `/cart/items?id=[id]&action=[action]` | `POST`  | Эндпоинт уменьшения/увеличения количества товара в корзине со страницы корзины |                                            |
| `/orders`                             | `GET`   | Эндпоинт получения страницы со списком заказов                                 |
| `/orders/{id}?newOrder=[newOrder]`    | `GET`   | Эндпоинт получения страницы заказа                                             |
| `/buy`                                | `POST`  | Эндпоинт совершения заказа                                                     |
